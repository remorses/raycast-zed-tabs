# Zed Tabs

A Raycast extension to list and switch between Zed editor window tabs.

## Features

- List all open Zed windows/tabs
- Search and filter tabs
- Switch to any tab with Enter
- Cached results for fast subsequent launches

## Installation

```bash
git clone https://github.com/remorses/raycast-zed-tabs
cd raycast-zed-tabs
npm install
npm run dev
```

## Requirements

- macOS
- Zed editor
- Accessibility permissions for Raycast (System Preferences > Privacy & Security > Accessibility)
